1. Indítsd el a **DockerDesktop** asztali alkalmazást!
2. Hozz létre egy mappát és nyisd meg **VSCode**-ban!
3. Hozd létre benne a **docker-compose.yaml** fájlt. A services alatt vannak a futtatható alkonténerek (pl. PHP, MySQL), a volumes-ban pedig a felhasznált mappák listája.
4. Nyisd meg a terminált!
5. Írd be a `docker compose up` parancsot! Az asztali alkalmazásban a Containers fület kell használni.
6. Ha le szeretnéd állítani a konténert, akkor a `docker compose down` paranccsal lehet.
7. Ha a docker-compose.yaml fájl tartalmát módosítod, akkor le kell állítani majd újra kell indítani a konténert. Új HTML, CSS, JS vagy PHP fájlok létrehozásakor ez nem szükséges.
